# ⚡ 电磁学仿真实验室

一个基于纯前端技术的交互式电磁学仿真平台，包含三个经典电磁学实验的完整可视化实现。

## 📋 项目概述

本项目使用 HTML5 Canvas + JavaScript 实现实时物理仿真，所有计算均在浏览器本地完成，无需后端支持。界面采用现代化设计，适合教学演示和录屏讲解。

## 📁 项目结构

```
electromagnetism_simulation/
├── index.html              # 主页面（单页应用入口）
├── style.css               # 统一样式文件
├── particle_deflection.js  # 模块1：带电粒子在复合电磁场中的偏转
├── capacitor_simulation.js # 模块2：平行板电容器动态变化仿真
├── solenoid_visualization.js # 模块3：通电螺线管空间磁感线可视化
└── README.md               # 本文档
```

## 📚 功能模块详解

### 模块1：带电粒子在复合电磁场中的偏转

#### 核心功能
- 三种模式切换：
  - **纯电场偏转**：演示带电粒子在匀强电场中的类平抛运动
  - **纯磁场圆周**：演示带电粒子在匀强磁场中的匀速圆周运动（洛伦兹力）
  - **正交电磁复合场**：速度选择器原理（当 qE = qvB 时粒子直线通过）

#### 交互控件
| 控件 | 功能 | 范围 |
|------|------|------|
| 电场强度 E | 调节电场力大小 | 0-100 N/C |
| 磁感应强度 B | 调节洛伦兹力大小 | 0-5 T |
| 粒子初速度 v | 调节粒子入射速度 | 10-100 (相对单位) |
| 粒子质量 m | 改变粒子质量影响加速度 | 1-20 (相对单位) |
| 电荷正负 | 切换正/负电荷 | + / - |

#### 物理实现
- 使用 **Verlet积分** 进行运动微分方程的数值求解
- 轨迹实时绘制，支持暂停、重置、清除轨迹
- 实时显示位置、速度、加速度数据

---

### 模块2：平行板电容器动态变化仿真

#### 核心功能
- 两种工作模式：
  - **接电源（U不变）**：电压恒定，电荷量随参数变化
  - **断电源（Q不变）**：电荷量恒定，电压随参数变化

- 实时计算并显示：
  - **电容 C** = ε₀εᵣS/d
  - **电压 U** = Q/C
  - **电荷量 Q** = C·U
  - **内部场强 E** = U/d

#### 交互控件
| 控件 | 功能 | 范围 |
|------|------|------|
| 极板间距 d | 调节两极板距离 | 1-50 mm |
| 极板面积 S | 改变极板正对面积 | 10-200 cm² |
| 相对介电常数 εᵣ | 插入介质改变电容 | 1-10 |
| 电源电压 U | 调节外接电源电压 | 1-100 V |

#### 可视化特性
- 动态绘制电场线，密度随场强变化
- 极板渐变色彩，正红负蓝
- 插入介质时显示介质区域
- 边缘效应的电场线弯曲效果
- 参数变化趋势高亮提示

---

### 模块3：通电螺线管空间磁感线可视化

#### 核心功能
- 基于 **毕奥-萨伐尔定律** 的磁场计算
- 截面磁感线分布图，展示磁场分布
- 鼠标悬停任意位置实时显示磁感应强度 B

#### 交互控件
| 控件 | 功能 | 范围 |
|------|------|------|
| 螺线管匝数 N | 改变线圈匝数 | 10-500 匝 |
| 电流大小 I | 调节通过的电流 | 0.1-10 A |
| 螺线管长度 L | 改变螺线管长度 | 5-50 cm |
| 螺线管半径 R | 改变螺线管半径 | 1-10 cm |
| 磁感线条数 | 控制绘制的磁感线数量 | 5-30 条 |
| 计算精度 | 数值计算精度设置 | 低/中/高 |

#### 物理公式
- **轴线中心**：B = μ₀nI （n = N/L 为匝数密度）
- **轴线端点**：B = ½μ₀nI （近似）

#### 高级特性
- 磁场方向动画演示
- N/S 极性标注
- 电流方向可视化（点/叉表示法）
- 磁场强度热图效果
- 十字光标精确定位

## 💻 技术实现要点

### Canvas 渲染优化
1. **分层绘制**：背景网格 → 场区域 → 场线 → 粒子/物体
2. **动画控制**：使用 `requestAnimationFrame` 实现流畅动画
3. **渐变效果**：`createLinearGradient` 和 `createRadialGradient` 增强视觉效果

### 数值计算方法
1. **Verlet积分**：粒子运动的稳定数值解法
2. **场线追踪**：基于梯度的磁感线/电场线生成算法
3. **实时更新**：参数变化立即反馈到计算和渲染

### 架构设计
- **模块化**：每个仿真独立为一个类，职责清晰
- **事件驱动**：所有控件通过事件监听触发更新
- **响应式**：Canvas 自适应窗口大小变化

## 🎯 使用建议

### 教学演示场景
1. **粒子偏转**：
   - 先展示纯电场，观察类平抛轨迹
   - 切换到纯磁场，观察圆周运动
   - 复合场模式：调节 v = E/B 观察直线通过（速度选择器）

2. **电容器**：
   - U不变模式：改变d/S，观察 C 和 Q 的变化
   - 切换到 Q不变模式：观察 E 保持恒定的特点
   - 插入介质（增大εᵣ），观察电容增大

3. **螺线管**：
   - 增大电流 I，观察磁感线变密
   - 增加匝数 N，观察中心B增大
   - 鼠标悬停对比轴线内外B的差异

### 录屏建议
- 建议分辨率：1920×1080
- 使用浏览器全屏模式（F11）
- 每个模块单独录制，讲解与操作同步
- 可配合物理公式讲解计算原理

## 🔧 扩展开发指南

### 添加新的物理量显示
在对应模块的 `updateDisplay()` 方法中添加：
```javascript
document.getElementById('yourElement').textContent = yourValue.toFixed(2);
```

### 自定义配色方案
修改 `style.css` 中的CSS变量或对应颜色值：
- 主色调：`#00d9ff` (青蓝), `#00ff88` (青绿)
- 警告色：`#ff6b6b` (红), `#ffc800` (黄)

### 添加新的仿真模块
1. 在 `index.html` 添加新的Tab和Canvas
2. 参考现有结构创建新的 `module_name.js`
3. 在Tab切换逻辑中添加初始化代码

## 📝 注意事项

1. **性能优化**：高精度计算和大量粒子可能影响低端设备性能，可降低精度设置
2. **浏览器兼容**：使用现代浏览器（Chrome 80+, Firefox 75+, Safari 13+）
3. **物理简化**：为了可视化效果，部分物理量进行了适当缩放，非实际物理尺度
4. **教学用途**：本仿真主要用于教学演示，精确计算请参考专业物理软件
